  ## About this issue

  We’re learning networks by building them—one tool at a time. Articles ship progressively; each is a self-contained lab with code, captures, and checks.

  **Early roadmap**
  - **01 · Packet Sniffer** — libpcap/BPF, capture/parse frames, filter by MAC/IP/port.
  - **02 · Ping** — craft ICMP echo, TTL hops, basic RTT stats.
  - **03 · Traceroute** — incremental TTL, parse ICMP time-exceeded, map the path.
  - **04 · Port Scanner** — TCP SYN, service banners, slow/stealth modes.
  - **05 · DNS Tool** — build a resolver; A/AAAA/CNAME, caching + `/etc/hosts`.
  - **06 · Tiny HTTP** — raw sockets, GET/POST; TLS hand-off to curl/openssl.
  - **07 · Wireshark Skills** — filters you’ll actually use; shareable captures.
